# sandoftime

#### 项目介绍
时间之沙
一秒时间可能对每个生命个体来说，意义不大，但所有人的时间加起来呢，在困境中的那一秒时间呢？
希望做一个与自然灾害相关的交流社区，个人可以从上面获取到面临灾难我们可以做哪些预防准备，或者在灾难后，我们可以怎么奉献自己的一份力量；而公益性组织也可以从上面获取到最新的灾难信息，哪些自愿者原意帮助受害人群；
这个社区希望在两者间建立一个友好，快速的信息交流渠道，争取把宝贵的时间用在救助受灾地。
